## 📊 技术架构

### 系统工作流

```
用户查询 → Claude Code (MCP) → 多源数据采集
                                    ↓
                            DeepResearch引擎
                                    ↓
                        报告生成 (Markdown/Word/PDF)
                                    ↓
                            人工审核优化 → 最终交付
```

### 核心技术栈

- **AI引擎**: Claude 3.5 Sonnet (Anthropic)
- **框架**: Claude Code + MCP (Model Context Protocol)
- **数据源**: Web搜索 + 学术数据库 + 行业报告
- **输出格式**: Markdown, Word, PDF
- **部署方式**: Docker容器化，支持云端/本地
